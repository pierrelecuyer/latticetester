Detailed Description

This file offers functions to generate random numbers use when making random searches.

The base generator is the 64-bit LFSR258 from [8], with period near \(2^{258}\), for 64-bit computers, and the 32-bit generator LFSR113 from [8], with period length near \(2^{113}\) on 32-bits computers. Thus the random numbers are different on 32-bits and 64-bits machines.
